# AI Project Planning Agent

An AI-powered project planning system built using CrewAI.

## Features

- Project Breakdown
- Time Estimation
- Resource Allocation
- Milestone Generation
- Structured Output using Pydantic

## Tech Stack

- Python
- CrewAI
- OpenAI
- Pydantic
- YAML
- Jupyter Notebook

## Installation

```bash
git clone https://github.com/yourusername/ai-project-planner.git

cd ai-project-planner

pip install -r requirements.txt
```

## Setup

Create .env

```env
OPENAI_API_KEY=your_key
```

## Run

```bash
python main.py
```

## Example Output

- Task List
- Milestones
- Resource Allocation Plan
